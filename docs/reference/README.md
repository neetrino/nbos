# Reference Docs

> Technical references and reusable engineering material.

This folder is not product canon. Product scope, business rules and UI behavior live in `docs/NBOS`.

Use this folder for:

- security and quality checklists;
- technology notes;
- templates;
- workflow examples;
- payment provider integration references.

## Payment Integrations

Payment references live in:

```text
docs/reference/payment-integrations/
```

Structure:

- `payments/` — internal integration guides and testing notes;
- `official-docs/` — original provider documents.

When implementing finance/payment flows, first read the NBOS Finance canon, then use these references for provider-specific technical details.
