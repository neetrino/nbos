# Payment Integrations Reference

> Provider-specific technical reference for payment integrations.

This folder is not the Finance business canon. Finance behavior, invoice/payment lifecycle and subscription rules live in:

```text
docs/NBOS/02-Modules/04-Finance/
```

Use this folder only for implementation details of payment providers.

## Structure

- `payments/` — internal integration guides, common rules and testing notes.
- `official-docs/` — original provider documents, PDFs, DOCX, HTML exports and related assets.

## Reading Order

1. Read the NBOS Finance canon.
2. Read `payments/00-INDEX.md`.
3. Read the provider-specific integration guide.
4. Use `official-docs/` only when provider details are needed.
