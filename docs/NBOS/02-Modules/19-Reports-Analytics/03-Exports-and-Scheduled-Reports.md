# Exports and Scheduled Reports

> NBOS Reports / Analytics - export, scheduled reports, report files and delivery rules.

## Export types

Reports can support:

- CSV;
- XLSX;
- PDF;
- snapshot link;
- Drive file export.

Not every report needs every export type.

## Export rules

Every export must include:

- report name;
- period;
- filters;
- generated by;
- generated at;
- data freshness;
- definitions / formulas where relevant;
- source modules;
- warning if data is incomplete.

## Export jobs

Heavy exports should run asynchronously.

```text
ExportJob
  report_key
  format
  filters
  status
  requested_by
  requested_at
  completed_at
  file_asset_id
```

Export files are stored through Drive.

## Sensitive exports

Some exports are sensitive:

- payroll;
- salary;
- bonus;
- credentials/security;
- finance P&L;
- partner payouts;
- client finance data.

Sensitive exports require:

- permission check;
- audit event;
- possibly step-up confirmation later;
- Drive permissions;
- retention policy.

## Scheduled Reports

Scheduled reports send or generate recurring reports.

Examples:

- weekly company summary;
- monthly finance report;
- monthly marketing performance;
- weekly sales pipeline;
- delivery risk report;
- support SLA report;
- KPI period report.

## Scheduled report model

```text
ScheduledReport
  report_key
  name
  schedule
  recipients
  format
  filters
  enabled
  owner
  last_run_at
  next_run_at
```

## Delivery channels

Possible delivery:

- email later;
- internal notification;
- Drive generated file;
- Dashboard priority card for failed report;
- Messenger/internal channel later if needed.

## Failure handling

If scheduled report fails:

- create notification for owner;
- show in scheduled reports status;
- store error;
- allow rerun.

## Audit

Audit required for:

- export requested;
- export completed;
- export downloaded if sensitive;
- scheduled report created/changed;
- scheduled report recipients changed;
- sensitive report generated.

## Retention

Export files should not live forever by default.

Retention options:

- temporary export;
- monthly snapshot;
- finance period archive;
- security export with strict retention.

Drive owns file lifecycle, Reports owns export metadata.
