# NBOS Archive

Эта папка хранит исторические NBOS-документы, которые больше не считаются активным каноном.

Использование:

- читать только как historical context;
- не использовать как source of truth при конфликте с актуальными документами;
- приоритет всегда у `00-Documentation-Hub.md` и текущих документов в `docs/NBOS`.

Сюда вынесены:

- старые сводные brief-документы;
- ранние roadmap-файлы;
- исторические git-анализы и промежуточные архитектурные справки.
